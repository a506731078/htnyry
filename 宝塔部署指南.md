# 宝塔面板Java项目部署指南

## 1. 服务器环境要求

### 操作系统
- Linux (推荐 CentOS 7+ / Debian 10+ / Ubuntu 20+)

### 安装宝塔面板
```bash
# CentOS
yum install -y wget && wget -O install.sh http://download.bt.cn/install/install_6.0.sh && sh install.sh

# Ubuntu/Debian
wget -O install.sh http://download.bt.cn/install/install-ubuntu_6.0.sh && sudo bash install.sh
```

### 安装软件
在宝塔面板「软件商店」中安装：
- **Java项目管理器** (版本 1.5+)
- **Nginx** (用于反向代理)
- **MySQL** (如果数据库不在本机，需要安装)

### JDK 版本
推荐使用 **Tomcat 8.5** 或 **Tomcat 9**，包含 JDK 8

## 2. 数据库配置

### 创建数据库
1. 宝塔面板 → 「数据库」 → 「添加数据库」
2. 填写信息：
   - 数据库名: `hhhh` (或你配置的名称)
   - 用户名: `hhhh` (或你配置的用户名)
   - 密码: `hhhh123` (或你配置的密码)
3. 字符集: `utf8mb4`

### 导入数据库结构
1. 找到数据库，点击「导入」
2. 上传 `sql/htny_complete.sql` 文件
3. 执行导入

## 3. 上传项目文件

### 创建网站目录
1. 宝塔面板 → 「网站」 → 「添加站点」
2. 填写域名或使用IP
3. 网站目录设置为项目部署目录，如: `/www/wwwroot/lis`

### 上传文件
将以下文件上传到 `/www/wwwroot/lis` 目录：

```
/www/wwwroot/lis/
├── lis-admin.jar      # Java主程序
├── config.json        # 配置文件
└── web/
    └── api.php        # PHP API文件
```

## 4. 配置Java项目

### 方式一：使用宝塔Java项目管理器

1. 打开「Java项目管理器」
2. 点击「添加Java项目」
3. 配置：
   - 项目名称: `lis`
   - 项目类型: `Spring Boot`
   - jar包路径: `/www/wwwroot/lis/lis-admin.jar`
   - 端口: `8808` (或config.json中配置的值)

4. 环境变量配置（可选，用于覆盖config.json）:
   ```
   CONFIG_PORT=8808
   CONFIG_DB_NAME=hhhh
   CONFIG_DB_USER=hhhh
   CONFIG_DB_PASSWORD=hhhh123
   CONFIG_ADDRESS=0.0.0.0
   ```

5. 启动命令:
   ```bash
   java -jar -Xmx1024M -Xms256M lis-admin.jar
   ```

### 方式二：使用命令行启动

SSH登录服务器，执行：

```bash
cd /www/wwwroot/lis
nohup java -jar -Xmx1024M -Xms256M lis-admin.jar > app.log 2>&1 &
```

### 方式三：创建systemd服务

创建服务文件:
```bash
vi /etc/systemd/system/lis.service
```

内容:
```ini
[Unit]
Description=LIS Logistics System
After=network.target mysql.service

[Service]
Type=simple
User=root
WorkingDirectory=/www/wwwroot/lis
ExecStart=/usr/bin/java -jar -Xmx1024M -Xms256M lis-admin.jar
Restart=always

[Install]
WantedBy=multi-user.target
```

启用服务:
```bash
systemctl daemon-reload
systemctl enable lis
systemctl start lis
systemctl status lis
```

## 5. 配置Nginx反向代理

1. 宝塔面板 → 「网站」 → 点击你的网站 → 「设置」
2. 选择「反向代理」 → 「添加反向代理」
3. 配置:
   - 代理名称: `lis-api`
   - 目标URL: `http://127.0.0.1:8808`
   - 发送域名: `$host`
   - 内容替换: (留空)

## 6. 放行端口

### 宝塔面板放行
1. 「安全」 → 「防火墙」
2. 添加规则: `8808` 端口允许

### 服务器防火墙放行
```bash
# CentOS
firewall-cmd --permanent --add-port=8808/tcp
firewall-cmd --reload

# Ubuntu/Debian
ufw allow 8808/tcp
```

## 7. 验证部署

### 检查服务状态
```bash
# 查看Java进程
ps -ef | grep lis-admin.jar

# 查看端口占用
netstat -tlnp | grep 8808
```

### 访问测试
```
http://你的服务器IP:8808
```

### API测试
```
http://你的服务器IP:8808/api.php?dh=物流单号
```

## 8. 常见问题排查

### 问题1: 服务启动成功但无法访问
- 检查端口是否正确放行
- 检查防火墙设置
- 检查 server.address 配置（应为 0.0.0.0）

### 问题2: 数据库连接失败
- 检查数据库配置是否正确
- 检查数据库用户权限
- 检查数据库是否允许远程连接

### 问题3: 日志查看
```bash
# 实时查看日志
tail -f /www/wwwroot/lis/logs/*.log

# 查看最新错误日志
tail -100 /www/wwwroot/lis/logs/sys-error.log
```

### 问题4: 内存不足
- 调整 JVM 参数 `-Xmx` 和 `-Xms`
- 建议: `-Xmx1024M -Xms256M` (可根据服务器内存调整)
