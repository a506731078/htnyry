# 本地打包环境要求

## 1. JDK 环境

- **版本**: JDK 1.8 (Java 8) 或更高版本
- **下载地址**: https://www.oracle.com/java/technologies/downloads/#java8
- **环境变量配置**:
  ```
  JAVA_HOME = JDK安装路径（如：D:\BtSoft\jdk\jdk1.8）
  PATH = %JAVA_HOME%\bin;%PATH%
  ```

## 2. Maven 环境

- **版本**: Maven 3.6+ 或更高版本
- **下载地址**: https://maven.apache.org/download.cgi
- **环境变量配置**:
  ```
  MAVEN_HOME = Maven安装路径（如：D:\BtSoft\apache-maven-3.8.6）
  PATH = %MAVEN_HOME%\bin;%PATH%
  ```

## 3. 验证环境

打开命令行窗口，执行以下命令验证：

```bash
java -version
# 应该显示 Java 1.8.x 版本

mvn -version
# 应该显示 Maven 3.6+ 版本
```

## 4. 项目打包步骤

### 方式一：使用打包脚本（Windows）

```bash
cd java
bin\package.bat
```

### 方式二：使用命令行

```bash
cd 项目根目录
mvn clean package -DskipTests
```

### 打包完成后

JAR包位置: `lis-admin\target\lis-admin.jar`

## 5. 常见问题

### Q: Maven 构建失败
- 检查网络连接（Maven需要下载依赖）
- 检查 JDK 和 Maven 版本是否正确

### Q: 打包成功但无法运行
- 检查 config.json 是否与 jar 包同目录
- 检查数据库配置是否正确
- 检查端口是否被占用
